¡Hola! Somos Blackcodes, un equipo de desarrolladores apasionados con una amplia experiencia en el emocionante mundo del desarrollo de aplicaciones. Trabajamos con una amplia gama de tecnologías backend como SQL Server, MySQL, MongoDB, Python y Docker, lo que nos permite crear aplicaciones eficientes y robustas. En el ámbito frontend, utilizamos tecnologías como HTML, CSS, JavaScript y React para desarrollar interfaces de usuario atractivas y funcionales. Además, contamos con conocimientos en desarrollo full stack, combinando nuestras habilidades tanto en backend como en frontend para ofrecer soluciones completas y de alta calidad. Aunque no hemos trabajado como desarrolladores en entornos reales, hemos aplicado estos conocimientos en proyectos personales, fortaleciendo nuestras habilidades en este campo.
